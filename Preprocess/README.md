# 공지

v3, v4의 특징
- 기존 v1은 시퀀스 단위로 학습하고 시퀀스 단위로 추론을 진행하였습니다. 그래서 v3, v4는 시퀀스 단위로 학습하되 웨이퍼 단위로 추론을 진행하도록 수정하였습니다.
- 각 모델을 학습시킬 때 TRACE-GPT_v1 파일을 참고하여 코드를 작성하는 것이 더 정확합니다.


v3, v4의 평가 과정(TRACE-GPT 기준)
1. 모델이 학습한 후, 정상 시퀀스와 불량 시퀀스를 추론함
2. 각 시퀀스에 대한 이상 점수를 출력 (1번 시퀀스: 10점, 2번 시퀀스: 20점, ..... 9번 시퀀스: 50점)
3. 이러한 시퀀스의 이상 점수를 Wafer_ID를 기준으로 묶음
4. 한 Wafer_ID 당 가장 높은 이상 점수를 선별
5. 해당 이상 점수를 기준으로 Wafer_ID의 정상/불량 판별 (이 과정에서 임계값이 형성되며 임계값을 기준으로 정상과 판별이 나뉨)
6. 이상 점수가 높으면 불량, 낮으면 정상으로 판별 (만약 정상 데이터인데 이상 점수가 높으면 오탐지)
